# Laundry Wahh

Laundry Wahh is a simple HTML webpage created to showcase a basic laundry service website.  
This project demonstrates fundamental HTML concepts such as headings, paragraphs, images, lists, tables, forms, and footer sections.

## Features

- Welcome heading for the website
- About section describing the laundry service
- Image representing laundry services
- List of services offered
- Price list displayed in a table
- Booking form for customers to enter their details
- Footer section

## Services Offered

- Washing and Drying
- Ironing and Folding
- Dry Cleaning
- Stain Removal
- Express Service
